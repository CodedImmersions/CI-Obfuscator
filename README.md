# CI Obfuscator
CI Obfuscator is a Unity Asset Obfuscator made for Unity 2020-6000.4+.
It renames GameObjects in scenes and a whole bunch of asset types, all to help reduce the risk of game copies.

## Platforms
Supported Editor Platforms:
- Windows
- macOS
- Linux

Supported Runtime Platforms:
- Windows
- macOS
- Linux
- iOS (untested)
- Android
